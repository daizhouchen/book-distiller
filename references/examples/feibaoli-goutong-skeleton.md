# 示例 · 非暴力沟通 骨架

基因：**模型(主) + 体验**

母题：**分离观察与评价**

记忆抓手：**把评价拆回观察**

## 这份骨架的关键点

1. **模型型书的结构专属**：不是人物剖面，是模型步骤分解
2. **适用/失效矩阵是核心**：模型型书最重要的就是**边界清晰**
3. **实操剧本必备**：让读者看到落地后的样子
4. **诚实暴露边界**：为什么在 PUA 下不适用，必须说
5. **系统逻辑 = 模型的哲学底座**：人本主义预设的暴露
