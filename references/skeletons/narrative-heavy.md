# 骨架 · 叙事基因主导（含史料型）

适用于以时间/情节/叙事为核心的书：人类简史、万历十五年、百年孤独（兼）、活着（兼）、三国演义、战争与和平、枪炮病菌与钢铁等。

## 模块顺序

1. 卷首 · 开篇 + 母题 + 记忆抓手
2. 时间轴全图（SVG）
3. 关键节点 × N
4. 多视角对照
5. 结构性归因 · 系统逻辑
6. 偶然与必然
7. 误读陷阱
8. 时代的茧
9. 金句精选
10. 思想坐标
11. 思考工具箱（折叠）

## 各模块内容要求

### 1 · 卷首

同通用规范。

### 2 · 时间轴全图

- 内嵌 SVG 时间轴
- 标出关键节点
- 按主题用不同轨道（如：政治 / 经济 / 文化 同时呈现）
- 朱砂标出决定性转折点

### 3 · 关键节点 × N

选 5-7 个决定性节点，每个按**三重结构**：

**事实层**（节点的具体构成）：
- 时间、地点、主要人物
- 事件序列
- 涉及的数据
- 事实矩阵（要素 × 来源）

**机制层**（为什么发生）：
- 偶然因素（如果没有 X，事件不会发生）
- 必然因素（结构性压力）
- 蝴蝶效应（后续影响链）

**观点层**：
- 作者对此节点的判断
- 现代学界的看法
- 读者能学到什么（结构性思维）

**原文对照**：作者的原文描述 × 我们的解读
**外部视角**：其他史家/学者对此节点的评价

### 4 · 多视角对照

⭐ 叙事型/史料型书的专属模块：

- 同一事件，站在不同人/阶层/地域的视角看会怎样？
- 作者的视角有什么偏好？他遮蔽了什么？
- 用表格/对照呈现：事件 × 多视角

例：鸦片战争 · 多视角
```
          清廷视角      英国视角      民间视角      国际视角
事件定位  通商纠纷      自由贸易      洋夷入侵      殖民扩张
```

### 5 · 结构性归因 · 系统逻辑

⭐ 核心章节：

- 整本书揭示的结构性逻辑是什么？
- 它是什么样的因果链/循环/分岔？
- 画出系统结构 SVG

例：人类简史的系统
```
  认知革命 → 想象共同体 → 合作规模扩大 → 统治世界
    ↓
  虚构能力变成了现代社会的基础设施
```

例：万历十五年的系统
```
  道德原则 vs 技术治理的根本冲突
    ↓
  官僚系统的"鱼死网破"均衡
    ↓
  表面的平静下结构性崩坏
```

### 6 · 偶然与必然

叙事型书避不开的深度追问：

- 哪些是结构必然？
- 哪些是偶然触发？
- 如果 X 不发生，整体结局会怎样？

### 7 · 误读陷阱

叙事型书容易被简化为"励志故事"或"阴谋论"——警示之。

## 密度要求

- 总字数：8000-15000
- 关键节点：≥ 5 个，每个 800+ 字
- 多视角对照：≥ 2 组
- 偶然/必然讨论：≥ 3 处
- SVG 图：≥ 3 张（时间轴 + 多视角矩阵 + 系统结构）

## 禁忌

- 不要流水账叙述 → 必须有结构化
- 不要只采纳作者视角 → 必须多视角对照
- 不要回避"如果……会怎样" → 反事实思维是叙事型书的深度所在
